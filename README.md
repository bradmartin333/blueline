# blueline

a fly fishing sim adapted from [paco-juego](https://bradmartin333.github.io/paco-juego/games/fishing/) with help from AI... this is really just a fun way for me to burn credits and test out new models / features

## controls

| key | action |
| --- | --- |
| `space` | context action (see below) |
| `m` | mend the line (during drift only) |
| `r` | reset / reel in (whenever the reset button is showing) |

### `space`, by game phase

| phase | action |
| --- | --- |
| idle | start the cast |
| casting | release the cast (time it!) |
| bite | set the hook |
| fight | strip line |
| reveal | continue (dismiss the catch) |

everything else (tackle/fly selection, tabs, rig changes) is mouse/click only
